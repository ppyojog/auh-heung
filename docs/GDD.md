## 1. 정체성 · USP
한 줄: **"약하고 쫓기던 존재가, 모두가 두려워하는 최강 포식자가 되는 권력 판타지."**

- 뱀서라이크의 *약→신* 성장 곡선이 이 변신 환상을 그대로 구현. 시작은 작고 포위당한
  느낌 → 점점 화면을 짓밟는 압도적 포식자.
- **차별점(클론에 없는 것):**
  1. 게임 이름값 하는 시그니처 **'어흥!' 궁극기**.
  2. 플레이어를 과장되게 떠받드는 **충신(忠臣) herald** — 성장할수록 칭송이 격해짐(무음 텍스트).
  3. 명작들의 장점 + **인디 정체성(영혼)**.
- **펫 타이니:** 작은 타이니가 이동 방향 뒤를 360° 졸졸 따라다니며 표정 반응(평소 🐯 / 처치·레벨업 😻 /
  피격 🙀) + **반투명 말풍선**으로 짧게 말함. 펫 감성으로 무해·귀여움·정서적 애착 강화.
- **타이니 메뉴(허브):** 스테이지 ±선택 · **내 빌드(스킬/패시브/레벨/체력) 상태** · 상점/업적 진입 · 포기 · (치트).
- **추가 컨셉 = 무해함·귀여움(코지 트렌드):** 강함(뽕)과 귀여움 공존. 코드 절차 애니메이션으로
  캐릭터가 숨쉬고·통통 튀고·눈 깜빡이고·꼬리 살랑(정적이지 않음). 캐릭터별 외형(백호 둥근귀/
  흑표 뾰족귀·점/무쇠뿔 뿔). 타이니는 귀엽고 무해하며 떠받드는 마스코트(등장 팝·통통).
- **수익화(욕 안 먹는 법):** 귀여운 **코스메틱 스킨** 중심(P2W·다크패턴 금지). 즉시 재시작 유지.
- **work-discreet:** 충신·연출이 전부 텍스트라 **무음으로도 뽕**. 몰래 하기 좋음.
- **사업:** 웹 자체 수익화는 약함 → 1차 '붙잡고 노는 무료 웹게임' → 모바일 포팅(광고/IAP).

## 2. 코어 루프
1. 타이틀에서 맹수(캐릭터) 선택 → **어흥! 생존 시작**
2. 고정 아레나. **드래그로 이동, 공격은 전자동**
3. 사방에서 의회 졸개가 몰려옴 → 자동 무기로 처치
4. 처치 → 마나 구슬 → 수집 → XP
5. 레벨업 → 일시정지 → **강화 3~4택**(신무기 / 강화 / 진화 / 패시브)
6. 처치로 **광기 게이지** 충전 → 가득 차면 **[어흥!]** 으로 화면 청소 + 광폭화
7. 시간 경과로 적 ↑ → HP 0이면 사망 → 기록 → 재도전 (엔드리스)

## 3. 조작 · UI
- 이동: 화면 어디든 드래그. **방향키는 하단 우측 고정·반투명**(오른손잡이), 노브가 진행 방향 반영.
- 공격: 전자동. 궁극기 **[어흥]** = 하단 좌측 원형 버튼 — **충전 전 반투명, 가득 차면 불투명+발광**.
- **타이니 호출 버튼**(🐯, 어흥 버튼 바로 위, **불투명**) → 일시정지 메뉴: 사냥터(STAGE) 선택 + 생존 정보 +
  내 빌드 + 전리품 상점 + **사냥터 이동**(±로 고른 STAGE로 옮겨 재개) + (치트). 게임 내 기능 허브.
  ※ '포기하고 마치기' 제거 — 어려우면 포기 대신 **STAGE를 낮춰 이동**(로그라이트 즉시-재시작 철학).
- 난이도 = **STAGE**: 초반 빠르게 오름(간격 13초~ → 갈수록 38초 cap)으로 *빠른 진행감*, 색으로 위협도.
  메뉴에서 ±로 목표 STAGE를 고른 뒤 **[이동] 버튼으로 확정**(도달한 최고까지). 낮추면 수월·올리면 보상↑.
  **한 칸의 난이도 간극은 완만**(diff=0.78+(stage-1)×0.07)하여 급상승 방지. maxStage는 슬롯별 영구 저장.
- 타이니 말풍선은 **반투명**(뒤 적 안 가림) + **표정(😺😼🔥😿🌟)으로 톤 전달**(안 읽어도 인식).
- 카메라 고정. 플레이어는 화면 안으로 클램프.

## 4. 시스템 요약 (수치는 `AuhHeung_DB.xlsx`)
- **단일 주인공 '호랑이가 되고 싶은 양'(🐑→🐯)**: 순한 양으로 시작 → 레벨/포식으로 성장할수록
  화면 속 모습이 **양→호랑이로 모핑**(tigerProg). 양털·둥근귀·순한눈 → 줄무늬 마스크('호랑이 탈')·
  뾰족귀·송곳니·사나운 눈매. USP(양→호랑이 권력 판타지)를 외형으로 직접 구현. 시작무기 발톱.
- **무기 5종**: 발톱·송곳니·포효·벼락·가시밭. 각 레벨 스케일.
- **무기 진화(핵심 훅)**: 최대 레벨 + 시너지 패시브 → 초월 무기(천 개의 발톱/죽음의 고리/대지진).
  레벨업 시 조건 충족하면 진화 선택지 우선 노출.
- **패시브 5종**: 야성(공격)·가죽(체력)·바람(이동)·굶주림(수집)·분노(공속).
- **장비 시스템(RPG, NEW)**: 3슬롯(무기🗡/방어구🛡/장신구💍) · 레어도 4단(일반·희귀·영웅·전설) ·
  스탯 보너스(공격%·공속%·체력·이동%·수집·재생). **영구 보유·장착**(슬롯별 localStorage).
  획득 = **보스 전리품 루트 드랍**(미보유 중 가중 랜덤) + **대장간(상태·장비 화면에서 송곳니 구매)**.
  가챠·다크패턴 없음(결정형, 윤리). 근거: Death Must Die/Halls of Torment식 슬롯 장비.
- **상태·장비 화면(NEW)**: 시작 스탯 미리보기(공격력·공속·체력·이동·수집·재생) + 장비 장착/구매 + (런 중)현재 빌드.
  타이니 메뉴 / 타이틀 / 사망 화면에서 진입.
- **시작 스테이지 선택(NEW)**: 타이틀·사망 화면에서 버튼으로 시작 사냥터 선택(도달한 최고까지).
  STAGE↑ = 적 강함 + **구슬 XP·전리품 비례↑**(빠른 성장 보상). XP배수 = 1+(stage-1)×0.2.
- **특별스킬 10종**: **10레벨마다 1개 선택(중복 불가)** — 판을 뒤집는 한 방.
  맹수의 격(공격+25%)·과부하(공속+30%)·강철가죽(피해−22%)·거미줄(적−18%)·굶주린포식(수집·XP)·
  재생(초당회복)·흡혈(회복3배)·연쇄폭발(처치시 폭발)·분신발톱(발톱+1)·서리손길(피격 적 둔화).
- **어흥! 궁극(광기)**: 처치로 충전 → 전 적 피해 + 넉백 + 5초 광폭화(공속·공격·이동 ↑).
- **충신 herald(타이니)**: 시작/레벨업/보스/궁극/위기/학살/마일스톤/변신마다 떠받드는 텍스트(무음).
  ※ 맥락 없는 잡담·랜덤 멘트 제거(대사는 상황에 맞는 것만).
- **적 8종(NEW)**: grunt·fast·**swarm(떼거리)**·tank·**splitter(분열)**·**bomber(자폭)**·
  **shooter(원거리 투사체)**·boss. STAGE가 오를수록 신종 해금(splitter S3·shooter S4·bomber S5).
  보스 90초 주기.
- **메타 진행(영구강화)**: 사망 시 플레이한 만큼 **송곳니(화폐)** 적립 → 타이틀 **전리품 상점**에서
  영구 강화 구매(공격/체력/이동/수집/송곳니획득). "죽음은 헛되지 않게" 무한 반복의 동력.
- **위협도 표시**: 현재 동적 난이도(`diff`)를 HUD에 등급(안온/평이/거셈/흉포/지옥)+색으로 표시.
- **포식(Devour)+성장**: 처치=삼킴 → 회복. ※ 크기는 거의 고정(1.0→1.08, tigerProg 기반) —
  무한정 커지지 않게 수정. 성장 표현은 '양→호랑이 모핑'이 전담.
- **난이도 하향(피드백 반영)**: diff=0.62+(stage-1)×0.055(clamp 0.5~2.4) · 적 체력/피해 계수↓ · 스폰 완화.
- **업적 8종**: 달성 시 송곳니 보상 + 슬롯별 영구 기록(🏆 화면, 사망 시 새 달성 표시).
- **코스메틱 스킨 6종**: 송곳니로 해금·착용(색만, P2W 없음 = 윤리적 수익화). 🎀 화면.
- **바닥 파워업 픽업(NEW)**: 처치 시 가끔 드랍(탱크·분열 확률↑). 가까우면 끌려오고 닿으면 발동 —
  🧲자석(화면 구슬 전부 흡수)·💣폭탄(전체 광역 피해)·❤회복(32%). 순간 변수·전술·도파민(근거: VS 바닥 아이템).
- **전리품 상자(NEW)**: 보스 처치 시 **무료 강화 선택 1회**(레벨업 화면 재사용). 보상 스파이크 = 도파민·빌드 가속.
  첫 보스 60초로 앞당김(초반 보상). 일반 보스 90초 주기.
- **스테이지 배경 테마(NEW)**: 3스테이지마다 분위기 전환(둥지→핏빛→한밤→보라→비취) + 부유 모트로 살아있는 배경.
  반복 맵의 단조로움 해소(근거: 장르 지루함 1순위 = 반복 웨이브·동일 레이아웃).
- **데일리 보너스**: 하루 첫 접속 송곳니 +30(비강제 리텐션). ※ 기존 세이브 슬롯에만 지급(빈 슬롯 악용 차단).
- **세이브 슬롯 3개**: 슬롯별 송곳니·강화·업적·스킨·기록 분리 저장(`surv_meta_N`/`surv_rec_N`,
  활성 `surv_slot`). 타이틀에서 슬롯 선택(요약 표시) + **세이브 삭제(슬롯 ✕ → 확인 다이얼로그)**.
- **레벨업 연출**: 레벨업 화면에 타이니가 떠받드는 표정+대사(뽕).
- **[치트]** 내부 테스트용 — 타이니 메뉴에 "+10 레벨"(자동 강화 10회). 출시 전 제거 예정.

## 5. 연출 · 게임필 (전부 코드)
- **아트:** 네온 벡터. 어두운 배경 + 발광 레이어. 플레이어=금빛 밝게 / 적=어두운 코어+네온 림으로
  대비 → 잡몹 더미 속 가독성 확보(클론 실패 1순위 원인 차단).
- **손맛:** 데미지 숫자, 화면 흔들림, 발광 파티클·잔상, 펄스 링, 피격 섬광, 보스 체력 링.
- **사운드:** Dart PCM 합성 → WAV → AudioElement. 효과음 7종 + 음소거.
