## 1. 정체성 · USP
한 줄: **"약하고 쫓기던 존재가, 모두가 두려워하는 최강 포식자가 되는 권력 판타지."**

- 뱀서라이크의 *약→신* 성장 곡선이 이 변신 환상을 그대로 구현. 시작은 작고 포위당한
  느낌 → 점점 화면을 짓밟는 압도적 포식자.
- **차별점(클론에 없는 것):**
  1. 게임 이름값 하는 시그니처 **'어흥!' 궁극기**.
  2. 플레이어를 과장되게 떠받드는 **충신(忠臣) herald** — 성장할수록 칭송이 격해짐(무음 텍스트).
  3. 명작들의 장점 + **인디 정체성(영혼)**.
- **추가 컨셉 = 무해함·귀여움(코지 트렌드):** 강함(뽕)과 귀여움 공존. 코드 절차 애니메이션으로
  캐릭터가 숨쉬고·통통 튀고·눈 깜빡이고·꼬리 살랑(정적이지 않음). 캐릭터별 외형(백호 둥근귀/
  흑표 뾰족귀·점/무쇠뿔 뿔). 타이니는 귀엽고 무해하며 떠받드는 마스코트(등장 팝·통통).
- **수익화(욕 안 먹는 법):** 귀여운 **코스메틱 스킨** 중심(P2W·다크패턴 금지). 즉시 재시작 유지.
- **work-discreet:** 충신·연출이 전부 텍스트라 **무음으로도 뽕**. 몰래 하기 좋음.
- **사업:** 웹 자체 수익화는 약함 → 1차 '붙잡고 노는 무료 웹게임' → 모바일 포팅(광고/IAP).

## 2. 코어 루프
1. 타이틀에서 맹수(캐릭터) 선택 → **어흥! 생존 시작**
2. 고정 아레나. **드래그로 이동, 공격은 전자동**
3. 사방에서 의회 졸개가 몰려옴 → 자동 무기로 처치
4. 처치 → 마나 구슬 → 수집 → XP
5. 레벨업 → 일시정지 → **강화 3~4택**(신무기 / 강화 / 진화 / 패시브)
6. 처치로 **광기 게이지** 충전 → 가득 차면 **[어흥!]** 으로 화면 청소 + 광폭화
7. 시간 경과로 적 ↑ → HP 0이면 사망 → 기록 → 재도전 (엔드리스)

## 3. 조작
- 이동: 화면 어디든 드래그(가상 조이스틱). 데드존 5px만 넘으면 즉시 풀스피드. 노브 반경 50px.
- 공격: 전자동. 궁극기: 우하단 **[어흥]** 원형 버튼(충전 링·가득 차면 발광). 음소거: 좌하단 🔊/🔇.
- 카메라 고정. 플레이어는 화면 안으로 클램프.

## 4. 시스템 요약 (수치는 `AuhHeung_DB.xlsx`)
- **캐릭터 3종**: 백호(균형)/흑표(유리대포)/무쇠뿔(탱크) — 시작 무기·스탯 상이.
- **무기 5종**: 발톱·송곳니·포효·벼락·가시밭. 각 레벨 스케일.
- **무기 진화(핵심 훅)**: 최대 레벨 + 시너지 패시브 → 초월 무기(천 개의 발톱/죽음의 고리/대지진).
  레벨업 시 조건 충족하면 진화 선택지 우선 노출.
- **패시브 5종**: 야성(공격)·가죽(체력)·바람(이동)·굶주림(수집)·분노(공속).
- **어흥! 궁극(광기)**: 처치로 충전 → 전 적 피해 + 넉백 + 5초 광폭화(공속·공격·이동 ↑).
- **충신 herald(타이니)**: 시작/레벨업/보스/궁극/위기/학살/마일스톤마다 떠받드는 텍스트 말풍선(무음).
- **적 4종**: grunt/fast/tank/boss. 시간(t)에 비례해 강해지고 종류 추가. 보스 90초 주기.
- **메타 진행(영구강화)**: 사망 시 플레이한 만큼 **송곳니(화폐)** 적립 → 타이틀 **전리품 상점**에서
  영구 강화 구매(공격/체력/이동/수집/송곳니획득). "죽음은 헛되지 않게" 무한 반복의 동력.
- **위협도 표시**: 현재 동적 난이도(`diff`)를 HUD에 등급(안온/평이/거셈/흉포/지옥)+색으로 표시.
- **저장**: localStorage — 최고 기록(`surv_rec`) + 메타(`surv_meta`: 송곳니·강화레벨).

## 5. 연출 · 게임필 (전부 코드)
- **아트:** 네온 벡터. 어두운 배경 + 발광 레이어. 플레이어=금빛 밝게 / 적=어두운 코어+네온 림으로
  대비 → 잡몹 더미 속 가독성 확보(클론 실패 1순위 원인 차단).
- **손맛:** 데미지 숫자, 화면 흔들림, 발광 파티클·잔상, 펄스 링, 피격 섬광, 보스 체력 링.
- **사운드:** Dart PCM 합성 → WAV → AudioElement. 효과음 7종 + 음소거.

## 6. 기술 · 배포
- Ticker(`createTicker`) dt 루프 + setState 리페인트. `CustomPaint`는 `RepaintBoundary`.
- 배포: `git push` → GitHub Actions(`subosito/flutter-action`) 빌드 → Pages.
  저장소 `github.com/ppyojog/auh-heung`. 빌드 실패해도 직전 라이브 유지.
- **검증:** 로컬 Flutter 미설치 → 수동(괄호·withValues 0·문법) 후 Actions 빌드로 확정.
- 향후: CanvasKit 전환(스웜 300+·maskFilter 글로우; 이모지 코드 교체 필요).

## 7. 코드 구조 (`lib/main.dart`)
